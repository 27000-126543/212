## 1. 产品概述
航天发射场综合调度与设备管理系统是为大型航天发射场打造的一体化指挥调度平台，覆盖发射任务从排程规划、任务推送、倒计时执行到事后统计分析的全生命周期管理。
- 解决多工位并行调度、设备资源冲突、气象窗口约束等复杂协同问题，确保发射任务安全高效执行
- 面向发射场指挥管理人员、技术操作人员和维保人员，实现发射流程数字化、调度智能化、管控可视化

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 总指挥 | 管理员分配 | 审批调整申请、中止/启动任务、查看全部数据 |
| 指挥员 | 管理员分配 | 管理工位任务、确认任务分配、查看排程 |
| 加注手 | 管理员分配 | 确认加注任务、上报加注状态、查看库存 |
| 测控操作员 | 管理员分配 | 确认测控任务、上报设备状态、查看占用 |
| 维保人员 | 管理员分配 | 接收维保工单、上报维修进度、扣减备件 |
| 系统管理员 | 系统初始化 | 管理用户、配置系统参数、导入基础数据 |

### 2.2 功能模块
1. **总览仪表盘**：关键指标卡片、任务状态分布、工位状态一览、最近告警、待办事项
2. **基础信息管理**：发射工位管理、火箭型号管理、有效载荷管理、发射窗口管理、人员管理
3. **智能排程调度**：排程参数配置、自动排程生成、排程甘特图、手动调整、气象约束集成
4. **任务推送与审批**：任务自动分配、岗位确认/申请调整、总指挥审批、消息通知中心
5. **发射执行管控**：倒计时流程控制、分系统状态校验、异常中止与备用方案切换
6. **实时监控告警**：任务状态追踪、关键参数监控（推力/轨迹/遥测）、超差报警、应急回收预案
7. **设备维保管理**：维保工单自动生成、维修班组分配、备件库存管理、维保记录查询
8. **统计分析报表**：发射成功率/故障率统计、人员工时统计、月度PDF报告导出
9. **可视化场区地图**：发射场平面图、工位实时状态、设备热力分布、动态标注

## 3. 核心流程

### 3.1 发射排程流程
用户录入基础信息（工位、火箭、载荷、窗口）→ 系统根据任务优先级、工位占用、设备库存、测控占用、气象约束自动生成排程 → 排程考虑工位转换时间≥7天和设备冷却复检周期 → 展示甘特图供审核 → 冲突检测通过后确认排程 → 自动推送任务到各岗位人员

```mermaid
flowchart TD
    "录入基础信息" --> "设置任务优先级与约束"
    "设置任务优先级与约束" --> "自动排程引擎计算"
    "自动排程引擎计算" --> "生成排程甘特图"
    "生成排程甘特图" --> "冲突检测"
    "冲突检测" -->|"无冲突"| "排程确认"
    "冲突检测" -->|"有冲突"| "调整排程参数"
    "调整排程参数" --> "自动排程引擎计算"
    "排程确认" --> "自动推送任务到岗位人员"
    "自动推送任务到岗位人员" --> "人员在线确认或申请调整"
    "人员在线确认或申请调整" -->|"确认"| "任务生效"
    "人员在线确认或申请调整" -->|"申请调整"| "总指挥审批"
    "总指挥审批" -->|"同意"| "调整排程参数"
    "总指挥审批" -->|"驳回"| "人员在线确认或申请调整"
```

### 3.2 发射执行流程
任务生效 → 进入准备阶段 → 倒计时启动 → 各阶段校验分系统状态 → 加注阶段 → 发射阶段 → 飞行阶段 → 再入阶段 → 若异常则中止并切换备用方案

```mermaid
flowchart TD
    "任务生效" --> "准备阶段"
    "准备阶段" --> "倒计时启动 T-2h"
    "倒计时启动 T-2h" --> "分系统状态校验"
    "分系统状态校验" -->|"全部通过"| "加注阶段"
    "分系统状态校验" -->|"异常"| "异常中止"
    "加注阶段" --> "加注完成校验"
    "加注完成校验" -->|"通过"| "发射阶段 T-0"
    "加注完成校验" -->|"异常"| "异常中止"
    "发射阶段 T-0" --> "飞行阶段"
    "飞行阶段" --> "实时参数监控"
    "实时参数监控" -->|"正常"| "再入阶段"
    "实时参数监控" -->|"超差"| "告警并启动应急回收"
    "异常中止" --> "切换备用方案"
```

### 3.3 设备维保流程
设备运行时长/发射次数达阈值 → 自动生成维保工单 → 分配维修班组 → 班组领用备件（扣减库存） → 完成维保 → 记录归档

```mermaid

    "分配维修班组" --> "领用备件"
    "领用备件" --> "扣减备件库存"
    "扣减备件库存" --> "执行维修"
    "执行维修" --> "维保记录归档"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深空蓝（#0A1628）为背景主色，搭配科技蓝（#00D4FF）作为强调色，警示橙（#FF6B35）和危险红（#FF3B3B）作为告警色
- 辅助色：星云紫（#7B61FF）用于特殊标注，翠绿（#00E676）用于成功/正常状态
- 按钮风格：圆角矩形（8px圆角），带微光边框效果，悬停时发光增强
- 字体：标题使用 Orbitron（科技感显示字体），正文使用 Noto Sans SC（中文优化），数据使用 JetBrains Mono（等宽数据字体）
- 布局风格：左侧固定导航栏 + 顶部状态栏 + 主内容区，卡片式布局，深色主题
- 图标风格：线性图标（Lucide），配合发光/呼吸动画效果
- 整体风格：深空科技风，暗色主题为主，带粒子背景和扫描线效果，营造航天指挥中心氛围

### 4.3 响应式设计
- 桌面优先设计，最小宽度1280px
- 导航栏在1280px以下自动折叠为图标模式
- 表格在窄屏下支持水平滚动
- 甘特图和地图保持固定宽高比缩放
